Control Commands in C: if and if-else
	if (condition){
	//do first thing
	}
	//do second thing

	if (condition){
	//do first thing
	}
	else{
	//do second thing
	}

Relational Operators for Conditional Statements:
	==	is equal to
	!=	is not equal
	>	is greater than
	>	is less than
	>=	is greater than or equal to
	<=	is less than or equal to
